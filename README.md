# Basic Gallery

Basic Gallery — это простое и удобное Android-приложение для просмотра фото и видео с вашего устройства. Поддерживает корзину, редактирование фотографий и многое другое — всё написано на Jetpack Compose.

## Что умеет приложение

**Галерея**
- Запрашивает разрешения на доступ к медиафайлам при первом запуске
- Загружает фото, видео и удалённые файлы из `MediaStore`, сортируя по дате съёмки
- Отображает медиа в сетке из трёх колонок с интерактивным скроллбаром-таймлайном справа — можно перетаскивать ползунок, чтобы быстро перемещаться по месяцам и годам
- Группирует фото по дням с понятными метками: «Сегодня», «Вчера» или дата
- При потягивании вниз появляется панель со счётчиком фото и видео

**Видео**
- Превью — первый кадр видео
- Значок длительности в углу: `ММ:СС` или `ЧЧ:ММ:СС` для длинных видео

**Полноэкранный просмотр**
- Листайте влево/вправо между всеми файлами текущего раздела
- Нажмите на экран, чтобы скрыть или показать панели управления
- Смахните вниз, чтобы закрыть
- Поддерживается зум для фотографий

**Редактор фотографий**
- Настройка выдержки, яркости, контраста и резкости
- Кадрирование с перетаскиваемыми угловыми маркерами
- При сохранении создаётся новый файл — оригинал остаётся нетронутым, дата съёмки сохраняется

**Корзина**
- Фото и видео из раздела «Фото» удаляются в корзину
- В корзине можно восстановить файлы или удалить их навсегда
- Многовыборочный режим с действиями «Восстановить» и «Удалить» в меню

## Стек технологий

- Kotlin
- Jetpack Compose + Material 3
- AndroidX ViewModel + StateFlow
- Coil (загрузка изображений)
- Gradle Kotlin DSL

## Требования

- Android Studio (актуальная стабильная версия)
- JDK 11
- Android SDK с `compileSdk 36`
- Минимальная версия Android: 9 (API 29)

## Сборка и запуск

```bash
./gradlew assembleDebug
./gradlew :app:installDebug
```

## Тесты и линтер

```bash
./gradlew connectedDebugAndroidTest

```

## Разрешения

Приложение запрашивает разрешения в зависимости от версии Android:

- Android 14+ (API 34+): `READ_MEDIA_IMAGES`, `READ_MEDIA_VIDEO`, `READ_MEDIA_VISUAL_USER_SELECTED`
- Android 13 (API 33): `READ_MEDIA_IMAGES`, `READ_MEDIA_VIDEO`
- Android 12 и ниже: `READ_EXTERNAL_STORAGE`

Корзина, восстановление и безвозвратное удаление используют MediaStore API, доступные с Android 11.
